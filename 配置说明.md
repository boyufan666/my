3. 运行 `npm run dev` 即可

1. 确认使用的是 WebSocket 地址（不是 HTTP）
2. 检查 Domain 是否为 `generalv1.5`
3. 验证 API 密钥是否正确
4. 查看服务器日志获取详细错误信息
